# Records

This directory stores historical evidence for security-significant or
production-impacting source changes in `openclaw-runtime-distribution`.

Use change records when a distribution-layer change affects:

- packaged runtime composition
- Telegram overlay or bundled runtime seams
- contract verification behavior against owner-published manifests
- stage/prod build reproducibility or delivery trust

For the structured record format, see
[change-records/README.md](change-records/README.md).
