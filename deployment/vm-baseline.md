# VM Baseline

## Recommended starting point

- Ubuntu Server LTS
- 2-4 vCPU
- 4-8 GB RAM
- 40+ GB disk
- NAT or host-only networking
- only the required application port forwarded
- Docker Engine installed inside the VM

## Why this baseline is used

This baseline is strong enough for a local isolated deployment while keeping the runtime separated from the main workstation.
